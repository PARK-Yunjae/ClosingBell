## 1️⃣ CCI 하드 필터

### 변경 내용

```
CCI 250 이상이면 무조건 TOP5 제외
(점수와 관계없이 필터링)
```

### 왜 추가했나?

최근 데이터 분석 결과 CCI 과열 종목의 익일 수익률이 저조:

| 날짜 | 종목 | CCI | 점수 | 결과 |
|------|------|-----|------|------|
| 1/13 | 한켐 | **578** | 86.9 | ❌ 제외 대상 |
| 1/16 | 아센디오 | **467** | 81.9 | ❌ 제외 대상 |
| 1/15 | 버넥트 | **479** | 82.4 | ❌ 제외 대상 |

### 코드

```python
# src/services/screener_service.py

CCI_HARD_LIMIT = 250

def filter_by_cci(scores, limit=CCI_HARD_LIMIT):
    """CCI 250+ 종목 필터링"""
    return [s for s in scores if s.score_detail.raw_cci <= limit]
```

## 3️⃣ 대기업 표시 (점수 가산 없음)

### 변경사항 (v6.2)

```
이전: 시가총액에 따라 점수 가산 (-1 ~ +5점)
이후: 점수 가산 없음, 대기업 표시만 함
```

### 표시 기준

| 시가총액 | 표시 |
|---------|------|
| 1조 이상 | 🏢 대기업 |
| 1조 미만 | - |

### 왜 변경했나?

- 순수하게 점수로만 TOP5 선정
- 대기업은 선택지로 표시만 제공
- 실제 매수 판단은 사용자가
